# Iris Flower Classification App

A machine learning web app that predicts the species of an Iris flower based on its measurements, built with Python, Scikit-learn, and Streamlit.


## Live App
(https://share.streamlit.io/ChiomaOnyeanula/Iris_app)

## Project Overview
This app uses a **Random Forest Classifier** trained on the classic Iris dataset to predict one of three flower species:
- Iris Setosa
- Iris Versicolour
- Iris Virginica

Users can input sepal and petal measurements using interactive sliders and receive an instant species prediction.

## Tools & Technologies
- **Python**
- **Scikit-learn** — Random Forest model
- **Streamlit** — interactive web interface
- **Pandas / NumPy** — data handling


## Dataset
The [Iris dataset](https://scikit-learn.org/stable/auto_examples/datasets/plot_iris_dataset.html) is a classic multiclass classification dataset containing 150 samples across 3 species, with 4 features:
- Sepal length (cm)
- Sepal width (cm)
- Petal length (cm)
- Petal width (cm)
